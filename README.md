# video_game_v1

Player presses start button.

Aspect formulas and starting values are randomly determined.
A_i,t = K * A_i,t-1 + beta_vector * x_vector

Sleep formula is randomly determined. (number of spoons gained is the sum of (scalar_i * an indicator function that takes value 1 if A_i is less than or equal to a value sleep_threshold_i)

Regulating costs 2 spoons. Regulating either decreases an aspect or does not change the aspect. If too much spoons are spent on regulating one asepct within a set of choices, hyperfocus/rumination occurs, which also occurs if the aspect gets too high.

how decreasing aspect A_i is determined.
